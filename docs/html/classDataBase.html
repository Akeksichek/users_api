Возвращает статическое подключение к базе данных.

Гарантирует создание только одного экземпляра подключения к базе данных mysql. Все последующие вызовы этой функции вернут тот же самый экземпляр подключения, что обеспечивает отсутствие дублирующихся подключений и оптимизацию производительности.

Parameters
addrАдрес базы данных (например, "localhost").
userИмя пользователя для аутентификации в базе данных.
passwdПароль пользователя для аутентификации в базе данных.
Returns
Указатель на объект DataBase, представляющий подключение к базе данных.
