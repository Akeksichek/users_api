Функция создания указателей на ключевые для всех контроллеров обьекты Boost.Beast request и Boost.Beast response.

По скольку контроллеры не возвращают готовый ответ, создаются указатели на уже полученный запрос и частично сгенерированный ответ на сервере и изменяют их в процессе своей работы
